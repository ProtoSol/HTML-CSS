About Thead, Tbody, and Tfoot

The <th> tag is used to define the headers of a table. It can be used with the colspan and rowspan attributes to define the size of the header.

The <tbody> tag is used to group the body content of a table. It is used with the <thead> and <tfoot> tags to divide the table into three parts: the table head, the table body, and the table foot.

The <tfoot> tag is used to define the footer of a table. It is used with the <thead> and <tbody> tags to divide the table into three parts: the table head, the table body, and the table foot.

Attributes of the Table Tag

The following are the attributes of the table tag:

  • border: specifies the border of the table. The value must be a number.
  • cellpadding: specifies the spacing between the cells of the table. The value must be a number.
  • cellspacing: specifies the spacing between the cells and the border of the table. The value must be a number.
  • width: specifies the width of the table. The value must be a number or a percentage.
  • height: specifies the height of the table. The value must be a number or a percentage.
  • align: specifies the alignment of the table. The possible values are left, center, right, and justify.

All these attributes should be changed using the style attribute. Thus, using CSS.
