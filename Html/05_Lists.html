Lists

Lists are used to define a group of related items in the document.

There are three list types in HTML: ordered, unordered, and definition lists.

Ordered Lists

Ordered lists are used to define a list of items with numbers or letters.

It is defined with the <ol> element and the items in the list are defined with the <li> element.

  1. First item
  2. Second item
  3. Third item

Unordered Lists

Unordered lists are used to define a list of items with bullets or other symbols.

It is defined with the <ul> element and the items in the list are defined with the <li> element.

  • First item
  • Second item
  • Third item

Definition Lists

Definition lists are used to define a list of items with definitions.

It is defined with the <dl> element and the items in the list are defined with the <dt> and <dd> elements.

First item
This is the first item
Second item
This is the second item
Third item
This is the third item

Nested Lists

Nested lists are used to define a list of items with sub items.

It is defined with the <ul> element and the items in the list are defined with the <li> element.

  • First item
  • Second item
    • Second item first sub item
    • Second item second sub item
      • Second item second sub item first sub sub item
      • Second item second sub item second sub sub item
  • Third item
