Heading elements are used to define headings in the document. They are defined with the <h1> to <h6> tags.

This is a paragraph.

<p> is a block level tag used to define a paragraph in the document.

<hr> is a block level tag used to create a horizontal line in the document.

Bold text

<b> is an inline tag used to define bold text in the document.

Italic text

<i> is an inline tag used to define italic text in the document.

Underlined text

<u> is an inline tag used to define underlined text in the document.

Strikethrough text

<strike> is an inline tag used to define strikethrough text in the document.

Superscript

<sup> is an inline tag used to define superscript in the document.

Subscript

<sub> is an inline tag used to define subscript in the document.

Font size 5

<font> is an inline tag used to define font properties in the document like font size, color, style, etc.

Arial font

The face attribute is used to define the font family in the document.

Red color

The color attribute is used to define the font color in the document.

We use block element first before the inline element. thus, we should always use paragraph tags before other tags like bold, italic, underline etc. only anchor tag is an inline element allowed contain block element tags.

Strong text

<strong> is an inline tag used to define strong text in the document. It is used to define text that has strong importance. (For Developers)

Emphasized text

<em> is an inline tag used to define emphasized text in the document. It is used to define text that has emphasis.

Inserted text

<ins> is an inline tag used to define inserted text in the document. It is used to define text that has been inserted in the document.

Code

<code> is an inline tag used to define code in the document. It is used to define code snippets.

        This is preformatted text.
        It is used to define preformatted text in the document.

<pre> is an block tag used to define preformatted text in the document. It is used to define text that has been formatted already.

HTML entities are special characters that are used to represent a single character in the document. They are used to define special characters like © for copyright, ® for registered, ™ for trade mark, etc.
