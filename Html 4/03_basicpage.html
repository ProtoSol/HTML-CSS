Target attribute in anchor tag
  • _self: Opens the linked document in the same frame as it was clicked (this is the default).
  • _blank: Opens the linked document in a new window or tab.
  • _parent: Opens the linked document in the parent frame.
  • _top: Opens the linked document in the full body of the window.
  • Custom Frame Name: Opens the linked document in a named iframe. Example: target="myFrame".
