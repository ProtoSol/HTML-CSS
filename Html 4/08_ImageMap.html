HTML Image Map

An image map is an image with clickable areas. The <map> tag is used to define a client-side image map. This map is linked to an image by using the usemap attribute of the <img> element.

Attributes of Image Map

  • name: Specifies the name of the image map.
  • coords: Specifies the coordinates of the map area.
  • shape: Specifies the shape of the map area (rect, circle, poly).

Map and Area Tag Attributes

The <map> tag is used to define an image map, which is a collection of clickable areas on an image. Each area within the image map is defined using the <area> tag. Below are the common attributes of the <map> and <area> tags:

Map Tag Attributes

  • name: Specifies the name of the image map. This attribute is used to associate the map with an image's usemap attribute.

Area Tag Attributes

  • alt: Provides alternative text for the area, which is displayed when the image is not available.
  • coords: Specifies the coordinates for the area. The shape of the area determines the number and meaning of the coordinates.
  • shape: Defines the shape of the area. Possible values are rect (rectangle), circle (circle), and poly (polygon).
  • href: Specifies the URL of the page the area links to.
  • target: Determines where to open the linked document. Common values include _blank (new window/tab), _self (same frame), _parent (parent frame), and _top (full window).
